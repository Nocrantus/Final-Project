Data from: Covid-19 Data
Project Started: 12/2/2025
Project Completed: 12/9/2025
Use the slider to select a date, and the dropdown box to either see cumulative covid cases or cumulative covid deaths. Then, press update to see the number around the map.
Click on a city, and you'll see a chart of cases or deaths over time in that town.

Note: Although many values were imputated in the original dataset, none of the values I used to create this visualization were imputated
